DG Automation (SIA&AIPL)

(DG250kVA Automation)

Initially, we observed that the DG (diesel generator) was operating in manual mode. We investigated the reason it wasn’t running automatically and discovered that an ATS (Automatic Transfer Switch) system was required. However, the cost of a standard ATS system was quite high—around ₹2 to ₹3 lakhs. To avoid the expense, we designed a custom automation solution using a PLC (Programmable Logic Controller) to control the DG's operation.
